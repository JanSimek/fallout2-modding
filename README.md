# Fallout 2 Modding Documentation

Technical documentation for Fallout 2 modding, covering SSL scripting and the [fallout2-ce](https://github.com/alexbatalov/fallout2-ce) engine internals.

## Contents

- **File Formats** - Binary data formats (.PRO, .MAP, .FRM, etc.)
- **SSL Scripting** - Script language syntax and built-in functions
- **Engine API** - Internal engine functions and data structures

## Development

### Prerequisites

- Node.js 18+
- npm

### Local Development

```bash
cd website
npm install
npm start
```

The site will be available at `http://localhost:3000`.

### Build

```bash
npm run build
```

## Deployment

The documentation is automatically deployed to GitHub Pages when changes are pushed to the `main` branch.
