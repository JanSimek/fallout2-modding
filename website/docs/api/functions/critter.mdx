# Critter Functions

Functions for working with critters (NPCs and creatures).

## opGetCritterStat

Get a critter's stat value.

### SSL Syntax

```ssl
variable value := critter_stat(critter, stat_id);
```

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `critter` | Object | The critter object |
| `stat_id` | int | Stat constant (see STAT_* constants) |

### Returns

The current value of the specified stat.

## opCritterHeal

Heal a critter.

### SSL Syntax

```ssl
critter_heal(critter, amount);
```

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `critter` | Object | The critter to heal |
| `amount` | int | HP to restore |

### Notes

- Cannot exceed maximum HP
- Does not affect dead critters

---

## opCritterDamage

Deal damage to a critter.

### SSL Syntax

```ssl
critter_dmg(critter, amount, damage_type);
```

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `critter` | Object | Target critter |
| `amount` | int | Damage amount |
| `damage_type` | int | Damage type flags |

### Damage Type Flags

| Flag | Description |
|------|-------------|
| `DMG_normal_dam` | Normal damage |
| `DMG_bypass_armor` | Ignore armor |
| `DMG_no_animation` | No damage animation |
| `DMG_knockout` | Knock out critter |

---

## opCritterInjure

Apply an injury to a critter.

### SSL Syntax

```ssl
critter_injure(critter, injury_flags);
```

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `critter` | Object | Target critter |
| `injury_flags` | int | Injury flags |

### Injury Flags

| Flag | Description |
|------|-------------|
| `DAM_CRIP_LEG_LEFT` | Crippled left leg |
| `DAM_CRIP_LEG_RIGHT` | Crippled right leg |
| `DAM_CRIP_ARM_LEFT` | Crippled left arm |
| `DAM_CRIP_ARM_RIGHT` | Crippled right arm |
| `DAM_BLIND` | Blinded |

## opGetCritterState

|-------|-------|

## opCritterAddTrait

Add a trait to a critter.

### SSL Syntax

```ssl
critter_add_trait(critter, trait_type, trait_id, value);
```

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `critter` | Object | Target critter |
| `trait_type` | int | Type of trait |
| `trait_id` | int | Specific trait |
| `value` | int | Trait value |

### Trait Types

| Type | Description |
|------|-------------|
| `TRAIT_PERK` | Add a perk |
| `TRAIT_OBJECT` | Object property |
| `TRAIT_TRAIT` | Character trait |
