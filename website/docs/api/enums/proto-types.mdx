# Prototype Type Enums

Constants for prototype (object) types.

import ImplLink from '@site/src/components/ImplLink';

## Object Types

The high byte of a PID indicates the object type:

| Value | Type | Description |
|-------|------|-------------|
| 0x00 | Item | Weapons, armor, consumables, etc. |
| 0x01 | Critter | NPCs and creatures |
| 0x02 | Scenery | Interactive environment objects |
| 0x03 | Wall | Blocking terrain |
| 0x04 | Tile | Floor/roof graphics |
| 0x05 | Misc | Exit grids, special objects |

### PID Structure

```c
// Extract type from PID
int type = (pid >> 24) & 0xFF;

// Extract proto number from PID
int proto_num = pid & 0x00FFFFFF;

// Build PID from type and number
int pid = (type << 24) | proto_num;
```
