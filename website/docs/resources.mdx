# Resources

A collection of useful resources for Fallout 2 modding and reverse engineering.

## Source Ports & Engine Reimplementations

### fallout2-ce (Community Edition)

The primary open-source reimplementation of the Fallout 2 engine in C++.

- [alexbatalov/fallout2-ce](https://github.com/alexbatalov/fallout2-ce) - Original repository by Alex Batalov
- [fallout2-ce Organization](https://github.com/fallout2-ce) - Community-maintained forks and related projects

### fallout-2-re

Reverse engineering efforts and documentation for the original Fallout 2 engine.

- [fallout-2-re](https://github.com/pfalcon/fallout-2-re) - Reverse engineering project

## Documentation & References

### F2MIC Documentation

Comprehensive scripting documentation and modding reference.

- [F2MIC GitHub Pages](http://f3mic.github.io/) - SSL scripting documentation and tutorials

## Community

## Tools

### Scripting

- [sslc](https://github.com/sfall-team/sslc) - SSL script compiler (Star-Trek Scripting Language Compiler)
- [int2ssl](https://github.com/falltergeist/int2ssl) - Decompiler for .int bytecode back to SSL source

### Editors

- **Mapper** - The official Fallout 2 map editor
- **F2wedit** - Item/critter prototype editor
- [sfall Script Editor](https://github.com/phobos2077/sfall_script_editor) - Script editor with syntax highlighting and integrated compiler
