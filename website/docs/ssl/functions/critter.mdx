# Critter Functions

Functions for managing critter statistics, traits, injuries, and state.

## critter_add_trait <span class="tag-vanilla">vanilla</span> {#critter_add_trait}

```ssl
int critter_add_trait(ObjectPtr who, int trait_type, int trait, int amount)
```

Adds a particular trait of a given type to a critter. Possible traits under the SPECIAL system include Perks, Traits, and Object-instance information (team numbers, AI packet numbers, etc.).

**Parameters:**
- `who` - Critter object
- `trait_type` - Trait type constant
- `trait` - Specific trait constant
- `amount` - Amount to add

## critter_attempt_placement <span class="tag-vanilla">vanilla</span> {#critter_attempt_placement}

```ssl
int critter_attempt_placement(ObjectPtr who, int hex, int elev)
```

Attempts to place a critter at a given destination hex and elevation. If placement fails, it tries to find a nearby hex as close as possible to the target.

**Parameters:**
- `who` - Critter to place
- `hex` - Target hex number
- `elev` - Target elevation (0-2)

## critter_inven_obj <span class="tag-vanilla">vanilla</span> {#critter_inven_obj}

```ssl
ObjectPtr critter_inven_obj(ObjectPtr who, int where)
```

Returns a pointer to an object in a given slot (NULL if none). Valid values for `where` are: `INVEN_TYPE_WORN`, `INVEN_TYPE_RIGHT_HAND`, and `INVEN_TYPE_LEFT_HAND`.

**Parameters:**
- `who` - Critter to check
- `where` - Inventory slot constant

**Returns:** `ObjectPtr` - Item in slot or NULL

**Example:**
```ssl
variable weapon;
weapon := critter_inven_obj(self_obj, INVEN_TYPE_RIGHT_HAND);
if (weapon) then begin
    display_msg("Critter is holding: " + obj_name(weapon));
end
```

## critter_mod_skill <span class="tag-vanilla">vanilla</span> {#critter_mod_skill}

```ssl
int critter_mod_skill(ObjectPtr who, int skill, int amount)
```

Modifies a given skill in a critter object by a given amount. The amount can be negative to decrease the skill.

:::note
Must be `dude_obj`. Will not work on other critters.
:::

:::tip
For tagged skills, the amount will be rounded down to the closest even number.
:::

**Parameters:**
- `who` - Must be `dude_obj`
- `skill` - <DefineRef name="SKILL_SMALL_GUNS" inline /> or other `SKILL_*` constant
- `amount` - Amount to modify (can be negative)

<FnRef fn="critter_mod_skill" />

## kill_critter <span class="tag-vanilla">vanilla</span> {#kill_critter}

```ssl
void kill_critter(ObjectPtr obj, int death_frame)
```

Kills a critter outright, placing it in the chosen death frame.

:::warning
This does NOT animate the critter and does NOT refresh the screen! It is meant to be used in scripts run when entering/exiting a map (map_init/map_exit).
:::

**Parameters:**
- `obj` - Critter to kill
- `death_frame` - Death animation frame
