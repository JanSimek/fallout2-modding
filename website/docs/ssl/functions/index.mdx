# SSL Function Reference

This section documents all built-in SSL functions available for Fallout 2 scripting.

## Function Categories

Functions are organized by their primary purpose:

| Category | Description |
|----------|-------------|
| [Script](./script) | Script control and context functions |
| [Object](./object) | Object creation, manipulation, and queries |
| [Critter](./critter) | Critter stats, traits, and state management |
| [Inventory](./inventory) | Item and inventory manipulation |
| [Animation](./animation) | Animation sequences and visual effects |
| [Combat](./combat) | Combat system functions |
| [Dialog](./dialog) | Dialogue system and conversation handling |
| [Map](./map) | Map loading, tiles, and spatial functions |
| [Time](./time) | Game time and timer functions |
| [Skill](./skill) | Skill checks and roll functions |
| [Party](./party) | Party member management |
| [Meta](./meta) | UI, settings, and game system functions |
| [Sfall](./sfall) | Sfall extension functions |

## Tags

Functions are tagged to indicate their origin:

- **vanilla** - Original Fallout 2 engine functions
- **sfall** - Extended functions added by sfall

## Common Types

### ObjectPtr

A pointer to a game object. Can be obtained from functions like `self_obj`, `dude_obj`, `source_obj`, or `target_obj`.

### int

A 32-bit signed integer value.

### string

A text string, typically enclosed in double quotes.

### void

Indicates the function does not return a value.
