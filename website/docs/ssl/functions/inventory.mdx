# Inventory Functions

Functions for managing inventories and manipulating items.

## add_obj_to_inven <span class="tag-vanilla">vanilla</span> {#add_obj_to_inven}

```ssl
void add_obj_to_inven(ObjectPtr who, ObjectPtr item)
```

Adds an object to another object's inventory.

:::note
This only works with objects of type Item.
:::

**Parameters:**
- `who` - Object to receive the item
- `item` - Item to add

**Example:**
```ssl
variable stimpak;
stimpak := create_object(PID_STIMPAK, 0, 0);
add_obj_to_inven(dude_obj, stimpak);
```

## inven_ptr <span class="tag-vanilla">vanilla</span> {#inven_ptr}

```ssl
ObjectPtr inven_ptr(ObjectPtr what, int slotNum)
```

Returns a pointer to the object in a specific slot number in a given object's inventory.

Macro alias for [`inven_cmds`](#inven_cmds)`(what, INVEN_CMD_INDEX_PTR, slotNum)`.

**Parameters:**
- `what` - Object to check
- `slotNum` - Slot index

**Returns:** `ObjectPtr` - Item in slot

**Example:**
```ssl
variable i, item_count, item;
item_count := inven_count(dude_obj);
for (i := 0; i < item_count; i := i + 1) begin
    item := inven_ptr(dude_obj, i);
    display_msg("Item " + i + ": " + obj_name(item));
end
```

## item_caps_adjust <span class="tag-vanilla">vanilla</span> {#item_caps_adjust}

```ssl
int item_caps_adjust(ObjectPtr obj, int amount)
```

Modifies the current caps count in an object by a given amount (positive or negative).

**Parameters:**
- `obj` - Object to modify
- `amount` - Amount to add (negative to subtract)

**Example:**
```ssl
// Give player 100 caps
item_caps_adjust(dude_obj, 100);

// Remove 50 caps
item_caps_adjust(dude_obj, -50);
```
