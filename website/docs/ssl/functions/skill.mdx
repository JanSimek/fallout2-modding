# Skill Functions

Functions for skill checks, dice rolls, and stat tests.

## has_skill <span class="tag-vanilla">vanilla</span> {#has_skill}

```ssl
int has_skill(ObjectPtr who, int skill)
```

Determines if a critter 'knows' a particular skill. Actually returns the level of the skill, which includes defaults.

**Parameters:**
- `who` - Critter to check
- `skill` - Skill constant

**Returns:** `int` - Skill level

## roll_vs_skill <span class="tag-vanilla">vanilla</span> {#roll_vs_skill}

```ssl
int roll_vs_skill(ObjectPtr who, int skill, int modifier)
```

Returns the value of a completed skill roll made upon a critter's skill level with a given modifier (may be zero).

This value may then be passed to `is_success` and `is_critical` to determine the appropriate states, and `how_much` can be used to determine the difference succeeded or failed by.

**Parameters:**
- `who` - Critter rolling
- `skill` - Skill constant
- `modifier` - Roll modifier (+/-)

**Returns:** `int` - Roll result

**Example:**
```ssl
variable roll_result;
roll_result := roll_vs_skill(dude_obj, SKILL_LOCKPICK, 0);

if (is_success(roll_result)) then begin
    if (is_critical(roll_result)) then begin
        display_msg("Critical success!");
    end else begin
        display_msg("Success!");
    end
end else begin
    display_msg("Failed by " + how_much(roll_result) + " points.");
end
```

## random <span class="tag-vanilla">vanilla</span> {#random}

```ssl
int random(int min, int max)
```

Returns a random value between min and max, inclusive.

**Parameters:**
- `min` - Minimum value
- `max` - Maximum value

**Returns:** `int` - Random number

**Example:**
```ssl
variable damage;
damage := random(5, 15);
display_msg("You deal " + damage + " damage!");
```
