# Fallout 2 Modding Documentation

Welcome to the comprehensive documentation for Fallout 2 modding, specifically focused on the [Fallout 2 Community Edition (fallout2-ce)](https://github.com/fallout2-ce/fallout2-ce) engine.

## What is this documentation?

This documentation covers:

- **File Formats** - Binary data formats used by Fallout 2 (`.PRO`, `.MAP`, `.FRM`, etc.)
- **SSL Scripting** - The scripting language used for game logic
- **Engine API** - Functions and data structures from the fallout2-ce source code

## Getting Started

### For Modders

If you're new to Fallout 2 modding, start with:

1. [Formats Overview](/formats/overview) - Understanding the game's data files
2. [SSL Overview](/ssl/overview) - Introduction to scripting
3. [Basic Examples](/ssl/examples/basic) - Simple script examples

### For Engine Developers

If you're contributing to fallout2-ce or want to understand the engine internals:

1. [API Overview](/api/overview) - Engine architecture
2. [Function Reference](/api/functions/critter) - Detailed function documentation
